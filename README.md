# 企微 & 个微 群发、开放接口

支持一个用户对多个企业微信进行聚合群发、聚合聊天、定时群发、用户动态分群、聊天消息记录等功能，每秒支持群发上万消息。



企微/个微托管在客户端，网页接口操作群发&在线聊天。



商务合作（开放源码，加V请备注来意）：veasion_wx
